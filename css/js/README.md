# Online Store 🛒

A beginner-friendly online store website built with HTML, CSS, and JavaScript.

## Features

✅ Product Catalog
✅ Shopping Cart
✅ Responsive Design
✅ Local Storage (cart persists)

## How to Use

1. Open `index.html` in your browser
2. Browse products and add to cart
3. Click cart icon to view items

## Files

- `index.html` - Homepage
- `cart.html` - Shopping cart
- `css/style.css` - Styling
- `js/app.js` - Cart logic
